## 2. Developer Adoption Funnel

```
Discover → Try → Use → Love → Advocate → Pay
   ↓        ↓     ↓      ↓       ↓         ↓
  HN/PH   Docker  Daily  Contrib  Blog    Enterprise
```

| Stage | Action | Content Needed | Metric |
|:--|:--|:--|:--|
| **Discover** | HackerNews, Product Hunt, Reddit, SEO | Landing page, comparison blog posts | Page views, GitHub visits |
| **Try** | `docker compose up` in 15 min | Quick Start guide, video tutorial | Docker pulls, GitHub clones |

## Key Principles

1. **Community-led growth first, enterprise sales second** — Build trust through open source
2. **Great docs = best marketing** — Developer tools live or die by documentation
3. **"Make it easy to start, hard to leave"** — 15-min Quick Start, then deep integrations
4. **SEA market first** — Home advantage in Thailand, then expand to Singapore, Vietnam
5. **Content, not ads** — Technical blog posts and tutorials drive qualified traffic
